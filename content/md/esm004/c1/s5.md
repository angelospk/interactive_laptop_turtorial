## Εισαγωγή

Η προσβασιμότητα του περιεχομένου που αναπτύσσεται μέσω εφαρμογών γραφείου εξασφαλίζει ότι άτομα με οπτικές, ακουστικές, κινητικές ή γνωστικές αναπηρίες μπορούν να χρησιμοποιούν αναγνώστες οθόνης, εργαλεία φωνητικής αναγνώρισης και άλλες υποστηρικτικές τεχνολογίες για να εργαστούν αποτελεσματικά με τα αρχεία Office.

Η ενότητα αυτή παρουσιάζει οδηγίες για την ανάπτυξη περιεχομένου το οποίο είναι προσβάσιμο: σχεδιασμός περιεχομένου, δομή περιεχομένου, και παρουσίαση περιεχομένου.

## Σχεδιασμός περιεχομένου

- Θα πρέπει να χρησιμοποιείται κατάλληλη γλώσσα και ορολογία ώστε το περιεχόμενο να είναι αντιληπτό.
- Το κείμενο θα πρέπει να αποδίδεται με σύντομες και κατανοητές προτάσεις.
- Το κείμενο θα πρέπει να συνοδεύεται από πολυμεσικό υλικό (αντιπροσωπευτικές εικόνες) και παραδείγματα.
- Τυχόν συντομεύσεις θα πρέπει να αναλύονται πλήρως την πρώτη φορά που εμφανίζονται στο κείμενο.
- Θα πρέπει να προτιμάται η ενεργητική φωνή έναντι της παθητικής φωνής.
- Χρησιμοποιήστε περιγραφικά ονόματα αρχείων.

---

## Δομή περιεχομένου

- Η δομή του περιεχομένου θα πρέπει να είναι σαφής και απλή.
- Το κείμενο θα πρέπει να έχει τίτλο
- Θα πρέπει να χρησιμοποιούνται κεφαλίδες και υπο-κεφαλίδες.
- Για μεγάλα κείμενα θα πρέπει να παρέχεται πίνακας περιεχομένων.

---

## Παρουσίαση περιεχομένου

- Δεν πρέπει να χρησιμοποιείται το χρώμα ως μοναδικός τρόπος μετάδοσης της πληροφορίας.
- Αν γίνεται χρήση χρώματος θα πρέπει να διασφαλίζεται η επαρκής αντίθεση μεταξύ φόντου και προσκηνίου.
- Αποφύγετε περίπλοκες εικόνες φόντου (backgrounds) που δυσκολεύουν την ανάγνωση.
- Η γραμματοσειρά που θα χρησιμοποιείται θα πρέπει να είναι απλή sans-serif.
- Στην περίπτωση των παρουσιάσεων χρησιμοποιήστε μέγεθος γραμματοσειράς τουλάχιστον 18pt.
- Θα πρέπει να υπάρχει κατάλληλη απόσταση μεταξύ των παραγράφων και των γραμμών μιας παραγράφου.
- Θα πρέπει να αποφεύγεται η χρήση πλάγιας γραμματοσειράς και υπογράμμισης (εάν δεν υπάρχει υπερσύνδεση).
- Το κείμενο θα πρέπει να είναι στοιχισμένο αριστερά.
- Δεν θα πρέπει να κόβονται οι λέξεις στο τέλος της γραμμής.
- Η χρήση λιστών βοηθά την κατανόηση του κειμένου.
- Οι εικόνες και τα γραφήματα θα πρέπει να συνοδεύονται από περιγραφικές λεζάντες.
- Οι εικόνες θα πρέπει να συνοδεύονται από εναλλακτικό κείμενο, εκτός και αν είναι διακοσμητικές.

### Το ήξερες ότι…;

Οι εφαρμογές γραφείου της Microsoft διαθέτουν ενσωματωμένο εργαλείο το οποίο μπορεί να ελέγξει το κείμενό σας για να εντοπίσει πιθανά προβλήματα προσβασιμότητας;

- Πηγαίνετε στο Review → Check Accessibility
- Διορθώστε όλα τα σφάλματα (Errors)
- Αντιμετωπίστε τις προειδοποιήσεις (Warnings)
- Εξετάστε τις συμβουλές (Tips)
