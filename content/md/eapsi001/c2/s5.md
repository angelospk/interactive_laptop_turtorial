## Υποενότητα: YouTube σε κινητό

### Δημιουργία λογαριασμού YouTube σε κινητό

Για να δημιουργήσετε λογαριασμό στο YouTube, χρειάζεστε έναν λογαριασμό Google. Το YouTube ανήκει στη Google, οπότε ο λογαριασμός Google είναι απαραίτητος για να συνδεθείτε, να σχολιάζετε ή να ανεβάζετε βίντεο.

### Σύνδεση σε λογαριασμό YouTube από κινητό

Αν διαθέτετε λογαριασμό Google, μπορείτε να συνδεθείτε από οποιαδήποτε συσκευή με πρόσβαση στο διαδίκτυο, ακολουθώντας τα παρακάτω βήματα:

- Ανοίξτε την εφαρμογή.
- Επιλέξτε «Εσείς» από το μενού πλοήγησης κάτω δεξιά στην οθόνη σας.
- Επιλέξτε «Σύνδεση».  
    
  *Εάν έχετε εισάγει σε προηγούμενο χρόνο τα στοιχεία ενός λογαριασμού επιλέξτε τον, διαφορετικά επιλέξτε «Προσθήκη λογαριασμού».*

- Συμπληρώστε το email σας.  
    
  *Εάν έχετε ξεχάσει τη διεύθυνση του email σας και έχετε δηλώσει αριθμό τηλεφώνου ανάκτησης ή εναλλακτικό email, μπορείτε να τα χρησιμοποιήσετε για να αποκτήσετε πρόσβαση.*

- Επιλέξτε «Επόμενο».
- Πληκτρολογήστε τον κωδικό πρόσβασής σας.  
    
  *Εάν έχετε ξεχάσει τον κωδικό πρόσβασής σας και έχετε δηλώσει αριθμό τηλεφώνου ανάκτησης ή εναλλακτικό email, μπορείτε να τα χρησιμοποιήσετε για να αποκτήσετε πρόσβαση.*

- Επιλέξτε «Επόμενο».
  - Την πρώτη φορά που θα συνδεθείτε, είναι πιθανό να σας ζητηθεί προαιρετικά να συμπληρώσετε στοιχεία που είχατε παραλείψει κατά την εγγραφή.
- Με τη σύνδεσή σας, θα μεταβείτε στην αρχική σελίδα του YouTube.

### Η αρχική σας σελίδα YouTube σε κινητό

Στην αρχική σελίδα του [YouTube](http://www.youtube.com), η διάταξη είναι σχεδιασμένη έτσι ώστε να βρίσκετε εύκολα βίντεο που σας ενδιαφέρουν.

### 1. Μενού πλοήγησης (στο κάτω μέρος οθόνης)

- Επιλέξτε «Αρχική», για να επιστρέψετε στην αρχική σελίδα με προτάσεις βίντεο.
- Επιλέξτε «Shorts», για να παρακολουθήσετε σύντομα βίντεο (κάθετης μορφής) διάρκειας έως 60 δευτερολέπτων.
- Επιλέξτε το κουμπί «Δημιουργία» (σύμβολο «+»):
  - Επιλέξτε «Βίντεο», για να ανεβάσετε ένα έτοιμο αρχείο βίντεο από τη συσκευή σας
  - Επιλέξτε «Ζωντανά», για να ξεκινήσετε live streaming (με κάμερα και μικρόφωνο).
  - Επιλέξτε «Short», για να τραβήξετε ή να ανεβάσετε ένα σύντομο βίντεο κάθετης μορφής (έως 60 δευτερόλεπτα).
- Επιλέξτε «Εγγραφές», για να βλέπετε νέα βίντεο από τα κανάλια στα οποία είστε εγγεγραμμένοι.
- Επιλέξτε «Εσείς» για να βλέπετε και να διαχειριστείτε το κανάλι σας, το ιστορικό προβολών, τις λίστες αναπαραγωγής, τα βίντεο που σας άρεσαν, τον χρόνο παρακολούθησης και τις προσωπικές σας ρυθμίσεις.

### 2. Επάνω μέρος της οθόνης

Στην εφαρμογή του YouTube στο κινητό, το επάνω μέρος της οθόνης περιλαμβάνει βασικές λειτουργίες πλοήγησης και χρήσης:

- Επιλέξτε το λογότυπο του YouTube (επάνω αριστερά), για να επιστρέψετε οποιαδήποτε στιγμή στην αρχική σελίδα.
- Πατήστε το κουδούνι «Ειδοποιήσεις», για να δείτε ενημερώσεις από τα κανάλια που παρακολουθείτε.
- Επιλέξτε το εικονίδιο του μεγεθυντικού φακού και πληκτρολογήστε λέξεις ή φράσεις στη γραμμή αναζήτησης, για να αναζητήσετε συγκεκριμένα βίντεο ή θέματα που σας ενδιαφέρουν.
  - Πατήστε το μικρόφωνο δίπλα στη γραμμή αναζήτησης, για να κάνετε φωνητική αναζήτηση.
- Επιλέξτε το εικονίδιο μετάδοσης και αν έχετε συμβατή συσκευή (π.χ. smart TV), μπορείτε να μεταδώσετε το βίντεο από το κινητό στην τηλεόρασή σας.
- Επιλέξτε το εικονίδιο της πυξίδας για εξερεύνηση σε κατηγορίες περιεχομένου όπως «Τάσεις», «Μουσική», «Αθλητικά» κ.λπ.
- Επιλέξτε «Θεματικές καρτέλες / Φίλτρα περιεχομένου» (κάτω από τη μπάρα): Μπορείτε να επιλέξετε κατηγορίες όπως: Όλα, Shorts, Μουσική, Παιχνίδια, Ειδήσεις, Ζωντανά για να προσαρμόσετε τις προτάσεις σας με βάση τα ενδιαφέροντά σας.

### Προφίλ χρήστη YouTube σε κινητό

Όταν είστε συνδεδεμένοι μπορείτε να βρείτε λεπτομέρειες για το προφίλ σας στην ενότητα «Εσείς» στο μενού πλοήγησής στο κάτω μέρος της οθόνης. Μπορείτε να δείτε και να διαχειριστείτε το κανάλι σας, το ιστορικό προβολών, τις λίστες αναπαραγωγής, τα βίντεο που σας άρεσαν, τον χρόνο παρακολούθησης και τις προσωπικές σας ρυθμίσεις.

- Επιλέξτε από την ενότητα «Εσείς» συγκεκριμένες λειτουργίες όπως:
  - Επιλέξτε «Φωτογραφία προφίλ» (αριστερά) - Ανοίγει το μενού του λογαριασμού σας, από όπου μπορείτε να:
    - Επιλέξτε να αλλάξετε τη φωτογραφία σας
    - Επιλέξτε να επεξεργαστείτε το όνομά σας
    - Επιλέξτε να επεξεργαστείτε το ψευδώνυμό σας
    - Επιλέξτε να δημιουργήστε, να δείτε ή επεξεργαστείτε το κανάλι σας
    - Αλλάξετε λογαριασμό
    - Εισέλθετε ή αποσυνδεθείτε
  - Επιλέξτε «Ιστορικό», για να δείτε βίντεο που παρακολουθήσατε πρόσφατα.
  - Επιλέξτε «Λίστα αναπαραγωγής», για να δείτε λίστες που έχετε δημιουργήσει ή αποθηκεύσει, ώστε να οργανώσετε τα αγαπημένα σας βίντεο ανά θέμα ή κατηγορία.
    - Επιλέξτε «Παρακολουθήστε αργότερα» για να δείτε ότι έχετε αποθηκεύσει για μελλοντική παρακολούθηση.
    - Επιλέξτε «Βίντεο που σας άρεσαν», για να βρείτε όλα τα βίντεο στα οποία πατήσατε το εικονίδιο «Μου αρέσει» 👍.
- Επιλέξτε «Τα βίντεο σας», για να έχετε πρόσβαση στο περιεχόμενο που έχετε ανεβάσει στο κανάλι σας.
- Επιλέξτε «Οι ταινίες σας», για να δείτε ταινίες και σειρές που έχετε αγοράσει ή ενοικιάσει.
- Επιλέξτε «Απόκτηση YouTube Premium», για να εγγραφείτε στην επί πληρωμή υπηρεσία χωρίς διαφημίσεις, με δυνατότητα λήψης βίντεο και αναπαραγωγής στο παρασκήνιο.
- Επιλέξτε «Χρόνος παρακολούθησης», για να δείτε πόσο χρόνο περνάτε παρακολουθώντας βίντεο στο YouTube.
- Επιλέξτε «Βοήθεια», για να βρείτε απαντήσεις σε συχνές ερωτήσεις ή να ζητήσετε υποστήριξη.
- Επιλέξτε «Σχόλια», για να στείλετε προτάσεις ή να αναφέρετε προβλήματα στη λειτουργία της πλατφόρμας.
- Επιλέξτε το εικονίδιο «Ρυθμίσεις» στο πάνω μέρος της οθόνης για να διαχειριστείτε τις ρυθμίσεις σας.

### Αναζήτηση περιεχομένου YouTube σε κινητό

Η αναζήτηση η οποία βρίσκεται στο πάνω μέρος της οθόνης σας επιτρέπει να πληκτρολογήσετε λέξεις σχετικές με αυτό που θέλετε να δείτε (π.χ. «παραδοσιακή μουσική», «μαγειρική συνταγή»). Μπορείτε να αναζητήσετε βίντεο, κανάλια ή λίστες αναπαραγωγής και να χρησιμοποιήσετε φίλτρα για να βρείτε πιο γρήγορα αυτό που σας ενδιαφέρει:

- Επιλέξτε το εικονίδιο αναζήτησης στο επάνω μέρος της σελίδας για να πληκτρολογήσετε λέξεις ή φράσεις.
  - Εναλλακτικά επιλέξτε το εικονίδιο με το μικρόφωνο που βρίσκεται δίπλα από τη φόρμα αναζήτησης για να πραγματοποιήσετε φωνητική αναζήτηση. Ενδέχεται να σας ζητηθεί άδεια για τη χρήση του μικροφώνου από το πρόγραμμα περιήγησης.
- Πατήστε «Εύρεση» για να δείτε τα αποτελέσματα. Στην περίπτωση της φωνητικής αναζήτησης όταν ολοκληρώσετε επιλέξτε το κόκκινο κουμπί με το μικρόφωνο.
- Αν θέλετε να περιορίσετε την αναζήτηση, μπορείτε να χρησιμοποιήσετε φίλτρα, όπως δεν έχουν προβληθεί, ζωντανά, ημερομηνία ανεβάσματος, διάρκεια, τύπο βίντεο κ.ά.

### Επανάληψη

Τα βήματα για αναζήτηση στο YouTube σε κινητό είναι:

1. Επιλέξτε το εικονίδιο αναζήτησης (μεγεθυντικός φακός) στο επάνω μέρος της εφαρμογής
2. Πληκτρολογήστε λέξεις-κλειδιά για βίντεο, κανάλια ή λίστες αναπαραγωγής
3. Πατήστε «Εύρεση» ή το κουμπί αναζήτησης στο πληκτρολόγιο για να δείτε αποτελέσματα
4. Επιλέξτε το μικρόφωνο για φωνητική αναζήτηση (μπορεί να ζητηθεί άδεια μικροφώνου). Όταν ολοκληρώσετε, πατήστε το κόκκινο κουμπί μικροφώνου για να σταματήσει
5. Επιλέξτε φίλτρα (π.χ. διάρκεια, ημερομηνία, ζωντανά) για πιο στοχευμένα αποτελέσματα

### Επανάληψη

Τα βήματα για παρακολούθηση και αλληλεπίδραση με YouTube Βίντεο σε κινητό είναι:

**Παρακολούθηση Βίντεο**

1. Επιλέξτε την εικόνα προεπισκόπησης (thumbnail) ή τον τίτλο του βίντεο για να ξεκινήσει η προβολή
2. Επιλέξτε το κουμπί «Αναπαραγωγή» (Play) στο κέντρο του βίντεο για να ξεκινήσει η αναπαραγωγή
3. Επιλέξτε «Παύση» (Pause) στο κέντρο του βίντεο αν θέλετε να διακόψετε προσωρινά και να συνεχίσετε αργότερα από το ίδιο σημείο
4. Επιλέξτε τα εικονίδια «Επόμενο» ή «Προηγούμενο» στο κέντρο του βίντεο για να αλλάξετε βίντεο στη σειρά αναπαραγωγής
5. Χρησιμοποιήστε τα φυσικά κουμπιά έντασης ήχου της συσκευής σας για να αυξομειώσετε την ένταση ή να κάνετε σίγαση
6. Σύρετε το κόκκινο σημείο στη γραμμή προόδου του βίντεο για να μεταβείτε σε άλλο σημείο του βίντεο
7. Επιλέξτε το εικονίδιο «Αυτόματη Αναπαραγωγή» για να ενεργοποιήσετε ή να απενεργοποιήσετε την αυτόματη προβολή του επόμενου βίντεο
8. Επιλέξτε το εικονίδιο «Συνδεσιμότητας» (cast) για να μεταδώσετε το βίντεο σε άλλες συσκευές (π.χ. smart TV, Chromecast) στο ίδιο δίκτυο
9. Επιλέξτε το γρανάζι «Ρυθμίσεις» και μετά «Ποιότητα» για να ρυθμίσετε την ανάλυση του βίντεο (π.χ. 480p, 720p, 1080p)
10. Επιλέξτε το εικονίδιο «Ελαχιστοποίηση» για να συνεχίσετε να βλέπετε το βίντεο σε μικρό παράθυρο, ενώ περιηγείστε στο YouTube
11. Επιλέξτε το τετράγωνο εικονίδιο «Πλήρης Οθόνη» κάτω δεξιά για να παρακολουθήσετε το βίντεο σε πλήρη οθόνη

**Αλληλεπίδραση με το Βίντεο**

1. Επιλέξτε το εικονίδιο «Μου αρέσει» ή «Δεν μου αρέσει» για να επηρεάσετε τις προτάσεις περιεχομένου
2. Επιλέξτε το εικονίδιο «Κοινοποίηση» για να διαμοιραστείτε το βίντεο μέσω κοινωνικών δικτύων ή με σύνδεσμο
3. Επιλέξτε «Αναφορά» ή «Αποθήκευση» για να αναφέρετε το βίντεο ή να το προσθέσετε σε λίστα αναπαραγωγής
4. Επιλέξτε το όνομα του δημιουργού για να δείτε το κανάλι του ή επιλέξτε «Εγγραφή» για να τον ακολουθήσετε

**Σχολιασμός Βίντεο**

1. Επιλέξτε την ενότητα «Σχόλια» κάτω από το βίντεο για να δείτε σχόλια ή να προσθέσετε το δικό σας
2. Επιλέξτε «Προσθήκη σχολίου» για να ανοίξετε το πεδίο εισαγωγής σχολίου
3. Πληκτρολογήστε το σχόλιό σας και επιλέξτε το εικονίδιο αποστολής για να το δημοσιεύσετε ή πατήστε εκτός για ακύρωση
4. Μπορείτε να επιλέξετε «Μου αρέσει» ή «Απάντηση» σε άλλα σχόλια για να συμμετάσχετε στη συζήτηση
