# Μάθημα 4: Δημιουργία ψηφιακού περιεχομένου

## Ενότητα: Πολυμέσα

## Υποενότητα: Δημιουργία προσβάσιμων πολυμέσων

### Εισαγωγή

Η δημιουργία πολυμέσων που είναι προσβάσιμα σε όλους τους χρήστες, συμπεριλαμβανομένων των ατόμων με αναπηρία, είναι μια σημαντική πρακτική που βελτιώνει την εμπειρία χρήστη και ενισχύει την ισότιμη πρόσβαση στο ψηφιακό περιεχόμενο. Σύμφωνα με τις Οδηγίες του W3C (World Wide Web Consortium) και το πρότυπο WCAG (Web Content Accessibility Guidelines), η ανάπτυξη προσβάσιμων πολυμέσων περιλαμβάνει συγκεκριμένες αρχές και πρακτικές που διασφαλίζουν ότι οι πληροφορίες είναι διαθέσιμες και κατανοητές για όλους τους χρήστες.

### Οδηγίες W3C και WCAG για προσβάσιμα πολυμέσα

Οι οδηγίες WCAG καθορίζουν μια σειρά από απαιτήσεις για τη δημιουργία πολυμέσων που είναι αντιληπτά, λειτουργικά, κατανοητά και αξιόπιστα. Κάποιες από τις πιο σημαντικές κατευθύνσεις περιλαμβάνουν τα εναλλακτικά κείμενα (alt text) και τις περιγραφές εικόνων καθώς και τη δημιουργία υποτίτλων και μεταγραφών για τα βίντεο.

### Εναλλακτικά Κείμενα (Alt Text)

Το εναλλακτικό κείμενο αποτελεί μια από τις πιο σημαντικές τεχνικές προσβασιμότητας για τις εικόνες. Πρόκειται για μια σύντομη, περιεκτική περιγραφή του οπτικού περιεχομένου που εμφανίζεται όταν η εικόνα δεν μπορεί να φορτωθεί ή όταν ένα πρόγραμμα ανάγνωσης οθόνης διαβάζει τη σελίδα σε χρήστες με προβλήματα όρασης.

**Οδηγίες για τη δημιουργία εναλλακτικών κειμένων**

- Να είναι περιγραφικά αλλά σύντομα, αποφεύγοντας περιττές πληροφορίες.
- Να μεταφέρουν την ουσία της εικόνας, χωρίς να ξεκινούν με φράσεις όπως «Εικόνα του...» ή «Φωτογραφία που δείχνει...».
- Εάν η εικόνα είναι διακοσμητική, χρησιμοποιείται η ιδιότητα `alt` με κενή τιμή «`alt=""`» για να παρακαμφθεί από το πρόγραμμα ανάγνωσης οθόνης.
- Σε περίπτωση που η εικόνα αποτελεί σύνδεσμο, το εναλλακτικό κείμενο πρέπει να εξηγεί τον προορισμό του συνδέσμου.

### Παράδειγμα

![Εικόνα με το δημοφιλές βρετανικό πιάτο γνωστό ως fish and chips](https://courses.nadia.gov.gr/assets/courseware/v1/0bca4845c73a3a30109e31b8f88ce2b8/asset-v1:EAPSI+esm004+2025_S2+type@asset+block/unit_4.2-image_08.webp)

Εικόνα 1: Το δημοφιλές βρετανικό πιάτο γνωστό ως "fish and chips"

Το εναλλακτικό κείμενο γι' αυτή την εικόνα δεν πρέπει να είναι: «πιάτο με φαγητό». Θα πρέπει να είναι μια συγκεκριμένη περιγραφή της εικόνας, όπως: «πιάτο με τηγανιτό ψάρι και πατάτες, ένα δημοφιλές βρετανικό πιάτο, με ένα ποτήρι μπύρα στο πλάι».

1. Το εναλλακτικό κείμενο πρέπει να περιλαμβάνει τα σημαντικά στοιχεία της εικόνας. Αυτά διαφέρουν ανάλογα με την εικόνα και το πλαίσιο στο οποίο χρησιμοποιείται. Για παράδειγμα, κατά περίπτωση, σημαντικό περιεχόμενο μπορεί να είναι το σκηνικό, τα συναισθήματα των ανθρώπων, τα χρώματα, ή μια σύγκριση μεταξύ δύο πραγμάτων.  
   Για παράδειγμα: «Δύο έξυπνα τηλέφωνα σε ένα τραπέζι. Το τηλέφωνο στα δεξιά έχει πιο πλατιά οθόνη, ενώ το τηλέφωνο στα αριστερά έχει κάμερα καλύτερης ποιότητας».
2. Μην ξεκινάτε το εναλλακτικό κείμενο με «μια φωτογραφία του…» ή «εικόνα του…»
3. Συνολικά το εναλλακτικό κείμενο θα πρέπει να είναι μια σύντομη πρόταση ή φράση, και δεν θα πρέπει να ξεπερνά τις δύο γραμμές κειμένου – ιδανικά έως 125 χαρακτήρες.

### Περιγραφές Εικόνων

Οι περιγραφές εικόνων είναι αναλυτικά κείμενα που προσφέρουν λεπτομερείς πληροφορίες για εικόνες που περιέχουν σύνθετο ή σημαντικό περιεχόμενο. Ενώ το εναλλακτικό κείμενο επικεντρώνεται σε μια σύντομη περιγραφή, οι περιγραφές εικόνων μπορούν να περιλαμβάνουν αναλυτική περιγραφή του τι απεικονίζεται, επεξήγηση δεδομένων σε γραφήματα ή διαγράμματα ή σημαντικά χρώματα και σχήματα που μεταφέρουν το νόημα σε μια εικόνα.

### Παράδειγμα

![άτομο εργάζεται σε φορητό υπολογιστή](https://courses.nadia.gov.gr/assets/courseware/v1/25ddacd8a896e7a6b82e4a92f3ed013a/asset-v1:EAPSI+esm004+2025_S2+type@asset+block/unit_4.2-image_09.webp)

Εικόνα 2: Άτομο εργάζεται σε φορητό υπολογιστή

Σε αντίθεση με το εναλλακτικό κείμενο, η περιγραφή της εικόνας πρέπει να είναι πιο αναλυτική.

Μια ενδεικτική περιγραφή για αυτή την εικόνα θα ήταν: «Ένας χώρος εργασίας με έντονο φυσικό φωτισμό. Στο επίκεντρο βρίσκεται ένα άτομο που εργάζεται σε φορητό υπολογιστή, πληκτρολογώντας με τα δύο χέρια. Στην οθόνη εμφανίζεται ένα γράφημα με μπάρες, που πιθανόν δείχνει στατιστικά ή δεδομένα. Δίπλα στον υπολογιστή υπάρχουν έντυπα με γραφήματα και πίνακες, καθώς και ένα σημειωματάριο με στυλό. Στο κάτω μέρος της εικόνας φαίνεται μέρος ενός κινητού τηλεφώνου. Το φόντο είναι θολό, δίνοντας έμφαση στη δραστηριότητα στο γραφείο.»

### Προσβάσιμα βίντεο

Η δημιουργία προσβάσιμου βίντεο προϋποθέτει ολοκληρωμένο σχεδιασμό από την αρχή της παραγωγής του.

Είναι απαραίτητα τα εξής στοιχεία:

**(Α) Υπότιτλοι** (απαιτείται για προσβασιμότητα σε επίπεδο Α). Για το κείμενο που θα χρησιμοποιηθεί στους υπότιτλους χρειάζεται να υπάρχει μέριμνα ώστε να είναι ευανάγνωστο (1) επιλέγοντας γραμματοσειρά sans serif (2) έχοντας κατάλληλο μέγεθος (3) φροντίζοντας να υπάρχει επαρκής αντίθεση μεταξύ του κειμένου και του φόντου. Αναλυτικές οδηγίες θα βρείτε εδώ: <https://www.w3.org/WAI/media/av/captions/>

**(Β) Περιγραφή της οπτικής πληροφορίας** (απαιτείται για προσβασιμότητα σε επίπεδο ΑΑ). Αν η ηχητική περιγραφή του βίντεο δεν είναι επαρκής για την πλήρη κατανόηση του περιεχομένου του βίντεο, θα πρέπει να παρέχεται επιπλέον περιγραφή, ώστε τυφλοί χρήστες να μπορούν να κατανοήσουν την οπτική πληροφορία του βίντεο. Αναλυτικές πληροφορίες παρατίθενται στον παρακάτω σύνδεσμο: <https://www.w3.org/WAI/media/av/description/>

**(Γ) Μεταγραφή της ηχητικής πληροφορίας** (απαιτείται για προσβασιμότητα σε επίπεδο Α), δηλαδή κειμενική απόδοση της ομιλίας και άλλης ηχητικής πληροφορίας. Αναλυτικές πληροφορίες θα βρείτε εδώ: <https://www.w3.org/WAI/media/av/transcripts/>

**(Δ) Νοηματική απόδοση για κωφούς χρήστες** (απαιτείται για πρόσβασιμότητα σε επίπεδο ΑΑΑ). Αν ενσωματωθεί παράθυρο νοηματικής, καλό είναι να μην κρύβει σημαντική πληροφορία από το βίντεο. Περισσότερες πληροφορίες θα βρείτε εδώ: <https://www.w3.org/WAI/media/av/sign-languages/>
