# Μάθημα 6: Επίλυση προβλημάτων

## Ενότητα: Συχνά προβλήματα και τρόποι επίλυσης

## Υποενότητα: Άλλα χρήσιμα εργαλεία επίλυσης προβλημάτων

### Εισαγωγή

Πέρα από τις βασικές μεθόδους που είδαμε, υπάρχουν και πιο εξειδικευμένα εργαλεία στα λειτουργικά συστήματα Windows και Mac, τα οποία μπορούν να βοηθήσουν στην διάγνωση και επίλυση προβλημάτων. Ας τα εξετάσουμε αναλυτικά.

### Διαγνωστικά εργαλεία των Windows/Mac

Τα λειτουργικά συστήματα παρέχουν ενσωματωμένα διαγνωστικά εργαλεία που ελέγχουν τη λειτουργία του υλικού και του λογισμικού και εντοπίζουν πιθανά προβλήματα.

**Στα Windows:**

- Τα **Windows** έχουν ενσωματωμένο εργαλείο που ελέγχει αυτόματα κοινά προβλήματα, όπως σύνδεση στο διαδίκτυο, ήχο, ενημερώσεις και άλλα.
- Στα Windows 11 μπορείτε να το βρείτε πηγαίνοντας στις Ρυθμίσεις → Σύστημα → Αντιμετώπιση προβλημάτων → Άλλα προγράμματα αντιμετώπισης προβλημάτων.
- Στα Windows 10 μπορείτε να το βρείτε πηγαίνοντας στο Ρυθμίσεις → Ενημέρωση & Ασφάλεια → Αντιμετώπιση προβλημάτων.
- Επιλέξτε το είδος του προβλήματος που αντιμετωπίζετε και ακολουθήστε τις οδηγίες που εμφανίζονται.

**Στο MacOS:**

- Το **Apple Diagnostics** (ή παλιότερα Apple Hardware Test) βοηθά στην ανίχνευση προβλημάτων υλικού.
- Για να το χρησιμοποιήσετε, κλείστε τον υπολογιστή Mac, πατήστε το κουμπί τροφοδοσίας και αμέσως κρατήστε πατημένο το πλήκτρο **D** μέχρι να ξεκινήσει το διαγνωστικό τεστ.
- Ακολουθήστε τις οδηγίες στην οθόνη για να δείτε αν υπάρχουν προβλήματα υλικού.

### Διαχείριση εργασιών (Task manager) - Windows / Παρακολούθηση δραστηριότητας (Activity monitor) – macOS

Όταν ο υπολογιστής σας λειτουργεί αργά ή κάποια εφαρμογή δεν ανταποκρίνεται, η χρήση της Διαχείρισης Εργασιών στα Windows ή του Activity Monitor στο Mac μπορεί να σας βοηθήσει να εντοπίσετε το πρόβλημα.

**Στα Windows - Task Manager:**

- Πατήστε ταυτόχρονα τα πλήκτρα **Ctrl + Alt + Delete** ή κάντε δεξί κλικ στη γραμμή εργασιών και επιλέξτε **Διαχείριση Εργασιών**.
- Στην καρτέλα **Διεργασίες** μπορείτε να δείτε ποιες εφαρμογές και διαδικασίες καταναλώνουν πόρους (CPU, μνήμη, δίσκο).
- Αν μια εφαρμογή «κολλάει», επιλέξτε την και πατήστε **Τέλος εργασίας** για να την κλείσετε.
- Στην καρτέλα **Εκκίνηση** μπορείτε να διαχειριστείτε τα προγράμματα που ξεκινούν μαζί με τον υπολογιστή.

**Στο MacOS - Activity Monitor:**

- Ανοίξτε το **Finder → Εφαρμογές → Βοηθήματα → Activity Monitor**.
- Παρακολουθήστε τη χρήση CPU, μνήμης, δίσκου και δικτύου από τις διάφορες εφαρμογές και διαδικασίες.
- Αν κάποια εφαρμογή δεν ανταποκρίνεται, επιλέξτε την και κάντε κλικ στο κουμπί **X** πάνω αριστερά για να τη κλείσετε.
- Το Activity Monitor βοηθά επίσης να εντοπίσετε ποια προγράμματα καταναλώνουν υπερβολικούς πόρους.
